GoModule is a tool for managing project dependencies. 
Before GoModule was introduced, 
there was no standard way to manage dependencies. 
Suppose you were using an open source package and the code was publicly available. 
After a while, an update was released for that package that included a lot of changes. 
You could determine that your project would not work with a specific version of GoModule. 
But using GoModule, you could use Unpacking and Your code package update was successful.
